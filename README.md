# HexLoader Client (Launcher)

Игровой клиент-лаунчер для HexLoader на базе Electron + React + TypeScript + TailwindCSS. Обеспечивает авторизацию, скачивание обновлений сборок и автоматический запуск игры.

## Функции
- **Синхронизация файлов**: Автоматически скачивает, обновляет и удаляет файлы модов на основе манифеста от бэкенда.
- **Поддержка версий Java**: Автоматически определяет и запрашивает нужную версию Java (указанную в манифесте сборки).
- **Прямой запуск**: Интеграция с процессом Minecraft, автоматическая передача токенов и запуск.
- **Настройка API**: Возможность сменить адрес бэкенда прямо в настройках интерфейса.
- **Интеграция с Discord RPC**: Отображение игрового статуса (при интеграции).

## Требования
- **Node.js**: версии 18+ (рекомендуется 20 LTS)
- **Java**: Наличие установленной Java 17/21 (в зависимости от требований запускаемой сборки).

## Быстрый старт (Разработка)

1. **Установка зависимостей** (из корневой папки клиента):
   ```bash
   npm install
   ```

2. **Запуск в режиме разработки**:
   ```bash
   npm run dev
   ```
   *Команда запустит Vite-сервер на порту 3000, дождется его готовности и откроет окно Electron.*

## Конфигурация API сервера

По умолчанию лаунчер ищет бэкенд на `http://127.0.0.1:4000`. Вы можете переопределить адрес бэкенда:
- Откройте окно лаунчера.
- Перейдите в настройки (иконка шестерёнки).
- В поле **«Адрес API Бэкенда»** введите нужный URL (например, `http://your-server-ip:4000`) и сохраните.
- Данные сохранятся в `localStorage` (для интерфейса) и в файл `client-settings.json` (для Electron-процесса лаунчера).

## Сборка дистрибутива (Production)

Для упаковки лаунчера в исполняемый `.exe` файл (для Windows):
```bash
npm run dist
```
После завершения процесса готовый установщик будет находиться в папке `dist/` (например, `HexLoader Setup 0.2.3.exe`).

## Структура проекта

- `electron/` — Код главного процесса Electron (работа с диском, сетью, запуск процессов игры).
  - `main.cts` — Точка входа главного процесса.
  - `launcher.cts` — Логика скачивания файлов и запуска Minecraft.
- `src/` — Интерфейс лаунчера (React).
  - `components/` — Визуальные компоненты UI (страница сборки, настройки, кнопки).
  - `lib/` — Вспомогательные библиотеки и моки API.
- `vite.config.ts` — Настройка сборщика Vite для рендерера.
